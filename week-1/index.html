CSS Nedir?

CSS (Cascading Style Sheet) açılımı “Basamaklı Stil Sayfaları” anlamına gelen web sayfalarında yazı tipi, renk, boyut, arka plan ve dokular gibi ögelerde arzu edilen değişiklikleri tanımlamanızı sağlayan bir programlama dilidir.

Float

CSS float özelliği web sayfalarında konulandırma ve düzen için kullanılır. Bir öğenin sayfa akışı içinde hangi yöne yaslanması gerektiğini ifade eder. Ancak bunu yaparken öğelerin görüntülenme varsayılanlarını (blok veya satır içi) etkiler, sayfa akışını değiştirir. Bir HTML öğesi float özelliği aldığında, öğeyi sayfa akışından çıkarır. Diğer öğeler ise float uygulanan öğenin varlığını görerek etrafından akışına devam eder. Bu çok yönlü çalışma şekli sayesinde oldukça fazla kullanım şekli ve alanı vardır. Alabileceği dört farklı değer vardır; left, right, none, inherit Bu örnek float kullanılarak yapılmıştır

Grid
